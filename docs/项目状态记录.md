## 项目目标

这是一个全屏个人主页项目，核心目标是：

- 展示强视觉的个人主页
- 页面采用全屏分板块结构
- 主体围绕 `Textline` 名字、three.js 字符球、鼠标交互和背景氛围层展开
- 所有模块按板块和功能拆分，样式统一使用 Less

## 当前板块

当前前端主板块按 `App.tsx` 顺序如下：

1. `hero`
2. `about`
3. `skills`
4. `works`
5. `experience`
6. `playground`
7. `contact`

对应导航数据来自 [`src/data/siteData.ts`](../src/data/siteData.ts)。
